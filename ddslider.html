Inspired by old school arcade shooters, mixed with roguelike elements, Drill Dash is set in a post-apocalyptic world where mech-equipped drillers are launched from railgun platforms into a vast environment full of destructive raiders, haywire robots and mutated locals.

  • Created an Upgrade System using Steamwork Cloud
  • Implemented the Localization Architecture
  • Implemented the Ads using Admob
  • Worked on the UI/UX for the game
  • Added analytics using Unity Analytics and Firebase
  • Tools Used: C#, Unity 3D, Git, VS/VS Code, Admob, Unity Analytic, Firebase
