The player plays a robot trying escape the raising water. Run and wall jump to freedom while shooting at turrets in the way. This game was created during LSU Sheveport Game Jam 2019 where the theme was flood. Our team of 6 got 1st place.

  • Created the art for the UI in Gimp
  • Worked on the UI/UX for the game
  • Tools Used: C#, Unity 3D, Git, VS/VS Code, Gimp
